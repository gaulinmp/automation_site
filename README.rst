================================
Automation Site
================================

Flask app for sending commands to Broadlink RM Mini


Run
----------------
In shell type: ``$ python run.py`` and open localhost:5000 in your browser.

Shell
-----------------
In shell type: ``$ python shell.py`` to access all flask methods in a python instance.
